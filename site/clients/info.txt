1. В папке clients должны находится только папки с именами необходимых клиентов! (classic, simple, tech)
2. Каждая клиентская папка должна содержать файлы jinput.jar ; lwjgl_util.jar ; lwjgl.jar и папку natives. 
	Также при необходимости загрузки дополнительных файлов положите сюда архив client.zip
3. В папке natives должны находится архивы с именами каждой ОС ( windows.zip ; linux.zip ; macos.zip ; solaris.zip )
